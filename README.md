# RTnexen PPS Tool

一鍵 Push、Pull 和查看 Status 的 KiCad Git 工具。

One-click Git Push, Pull, and Status tool for KiCad PCB Editor.

## Usage / 使用方式

1. 開啟 KiCad PCB Editor
2. 在上方工具列找到 **PPS Tool** 圖示（藍色與橘色箭頭）
3. 依照需求選擇功能：
   - **↑ Push** — 將變更推送到遠端
   - **↓ Pull** — 從遠端拉取最新版本
   - **● Status** — 查看目前分支與變更狀態
4. Push 時需填入 commit message，其餘功能全自動完成

---

1. Open KiCad PCB Editor
2. Find the **PPS Tool** icon (blue and orange arrows) in the top toolbar
3. Select the desired action:
   - **↑ Push** — Push changes to remote
   - **↓ Pull** — Pull latest changes from remote
   - **● Status** — View current branch and changes
4. Only Push requires additional input (commit message); all other actions run automatically
